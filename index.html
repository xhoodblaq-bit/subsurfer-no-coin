Das Spiel

🎮 Wie es funktioniert

Du surfst endlos über die U-Bahn-Gleise. Sammle Münzen und spezielle Items, um deine Punktzahl zu erhöhen. Weiche Hindernissen aus - wenn du getroffen wirst (ohne Shield), ist Game Over!

💡 Tipps zum Gewinnen:

  • Sammle Münzen für einen Multiplikator
  • Power-Ups greifen - Shield schützt dich einmal
  • Das Spiel wird immer schneller - bleib konzentriert!
  • Nutze Rutschen um niedrige Hindernisse zu vermeiden

Über das Spiel

Subway Surfer - No Coin Edition ist ein Browser-basiertes Runner-Spiel, inspiriert vom beliebten Mobile Game Subway Surfers. Das Spiel wurde vollständig in HTML5, CSS3 und JavaScript entwickelt und läuft direkt in deinem Browser.

Alle deine Highscores werden lokal in deinem Browser gespeichert, sodass du immer deine beste Leistung sehen kannst. Das Spiel ist vollständig responsive und funktioniert perfekt auf allen Geräten - vom Desktop bis zum Smartphone.

🛠️ Technologie

HTML5 Canvas Vanilla JavaScript CSS3 Animations LocalStorage API Touch Events Responsive Design
