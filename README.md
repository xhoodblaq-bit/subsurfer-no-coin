# No Coin SS

Ein Browser-Runner-Spiel im Stil von Subway Surfer mit eigener Startseite, Spielseite und Admin-Builder-Ansicht.

## Website-Dateien
- [index.html](index.html) – Startseite und Landingpage
- [style.css](style.css) – Styling und Layout der Website
- [game.html](game.html) – Spieloberfläche
- [game.js](game.js) – Spiel-Logik, Steuerung und Gameplay
- [admin.html](admin.html) – Admin-Panel zum Weiterbauen und Steuern
- [admin.js](admin.js) – Admin-Befehle und Einstellungen

## So öffnest du die Website
1. Öffne [index.html](index.html) in einem Browser.
2. Oder starte einen lokalen Server, zum Beispiel mit `python3 -m http.server 8000`.
3. Danach kannst du die Seiten unter `http://127.0.0.1:8000/` aufrufen.

## Steuerung
- Links / Rechts: Pfeiltasten oder `A` / `D`
- Springen: Pfeil hoch, `W` oder Leertaste
- Rutschen: Pfeil runter oder `S`

## Features
- 3 Spuren-Runner mit Hindernissen, Münzen und Power-ups
- Highscore-Speicherung im Browser
- Touch-Steuerung für Mobilgeräte
- Admin-Panel zum Senden von Befehlen und Anpassen von Einstellungen
